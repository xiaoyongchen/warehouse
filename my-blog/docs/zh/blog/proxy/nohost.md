## nohost抓包神器使用

### 安装
```bash
sudo npm i -g @nohost/server --registry=https://r.npm.taobao.org
```
**查看是否按照成功**
```bash
n2 --version
```

**启动｜停止**
```bash
n2 start ｜ n2 stop
n2 restart ｜ n2 restart —-reset 重置管理员账号
```
